Bot commands

parameters with an asterisk are required, you can also pipe in parameters by replying to a message

Command Usage Description
%help (%commands, %cmds) %help gives a link to this page
%ping %ping just says the uptime of the bot
prefix? prefix? says what the current prefix is
%setprefix %setprefix *[prefix] change the prefix
%join (%part) %join join or part bot in your channel
%link (%unlink) %link *[lastfm|letterboxd] *[username] link or unlink your lastfm or letterboxd account
%letterboxd (%boxd, %lb) %letterboxd [username] returns some information about a letterboxd user's most recently logged film
%movie (%film, %mv) %movie *[query] returns some information about a movie on letterboxd
%rating (%rt, %review) %rating *[username] *[query] returns a users rating and review of a *logged* movie on letterboxd
%song (%s, %sc) %song [username] shows a lastfm user's most recently played song, %sc to include play count
%whoknows (%wk) %whoknows [artist] shows #plays of users that have lastfm linked for an artist
%fortune (%f, %cookie) %fortune daily fortune message with a streak if you do it everyday
%searchlogs (%sl) %searchlogs *[channel] *[user] *[query] search somebody's logs for something using bestlogs
%connections (%conn, %c) %connections [user] [platform] lists the potatbotat connections of a user
%listcmds (%listcmd, %lc) %listcmds [channel] returns a link to a channel's custom potatbotat commands
%namechange (%nc) %namechange [user] view a user's previous usernames
%newname (%nn) %newname [username] try to find a current username from an old one using supibot database
%unicode (%u) %unicode *[message] gives you a link to a website that lets you identify what unicode characters a message contains
